# Logo Prompts

## 核心方向

为 `Ditto Skill` 设计一个适合 GitHub 仓库头像和横幅的视觉形象。

这个形象应该让人联想到：

- 软糯
- 可变形
- 会模仿
- 有学习能力
- 带一点工程感

不要直接使用或模仿任何已有受版权保护的游戏角色或怪物形象。

## Prompt 1: GitHub 头像版

Design a clean logo for "Ditto Skill", an open-source tool that distills software repositories into reusable skills for coding assistants. Create a soft, shape-shifting blob mascot with a simple friendly face, subtle engineering cues, and a clear silhouette. It should feel adaptive, intelligent, playful, and trustworthy. Use a clean flat vector style, minimal color palette, transparent or white background, and make it suitable for a GitHub avatar. Do not imitate any copyrighted game creature or character.

## Prompt 2: 横幅 / Banner 版

Create a banner illustration for "Ditto Skill", an open-source repo distillation tool for coding assistants. Show a soft, morphing mascot studying commit graphs, code blocks, branching lines, and small skill icons, as if learning and transforming engineering patterns. Use a modern editorial tech-illustration style, clean composition, subtle gradients, strong readability, wide horizontal layout, and an open-source project aesthetic. Do not imitate any copyrighted character.

## Prompt 3: 图标优先版

Create a minimalist icon for "Ditto Skill": a rounded shape-shifting blob with a subtle branching mark, sparkle, or small diff-like symbol inside, suggesting adaptation, cloning, and skill transfer. Use a clean vector logo style, simple shapes, memorable silhouette, and transparent background. Suitable for favicon, GitHub avatar, and docs header. Do not imitate any copyrighted creature.

## Prompt 4: 更偏工程气质的版本

Create a modern mascot logo for "Ditto Skill", a repository distillation tool for coding assistants. The mascot should be a soft morphing creature combined with subtle engineering symbols such as commit nodes, branch lines, or modular blocks. Keep the design simple, iconic, open-source friendly, and highly recognizable. Avoid direct resemblance to any existing game or anime creature.

## 生成建议

如果你要实际生成 logo，建议：

1. 先生成头像版 logo
2. 再基于同一风格生成 banner
3. 优先要：
   - 简洁轮廓
   - 少颜色
   - 透明背景
   - 小尺寸下仍清楚
4. 避免：
   - 过于复杂的细节
   - 太像现成角色
   - 太重的 3D 光影
   - 太长的文字直接塞进图里
